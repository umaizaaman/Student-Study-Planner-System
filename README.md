 # 📚 Student Study Planner Management System

## **Project Overview**

Student Study Session Management System is a C++ based console application developed using Object-Oriented Programming (OOP) concepts.

This project helps students manage their academic activities by providing features like course management, assignment tracking, exam score management, study planning, and progress tracking.

The main goal of this project is to organize students' study records in a structured way and demonstrate the practical implementation of advanced OOP concepts in C++.

 👨‍🎓 Student Management

 📖 Course Management

## 🧠 OOP Concepts Implemented

This project demonstrates various Object-Oriented Programming concepts:

 ✅ Encapsulation
Data members are protected and accessed through appropriate member functions.

✅ Abstraction
Only necessary functionalities are exposed to users while hiding internal implementation details.

 ✅ Inheritance
Used to create relationships between different classes and reuse code.

✅ Multiple Inheritance
Implemented multiple parent-child class relationships.

✅ Virtual Inheritance
Used to handle inheritance ambiguity and improve class hierarchy.

✅ Polymorphism
Implemented runtime polymorphism using virtual functions.

✅ Classes and Objects
Designed different classes to represent students, courses, assignments, exams, and study sessions.
